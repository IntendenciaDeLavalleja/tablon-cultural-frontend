# Tablon Cultural - Demo Estatica

Demo de exhibicion creada con Astro + React sin backend.

## Comandos

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Contenido

- Cuatro eventos mock cargados desde `src/data/events.ts`
- Imagenes locales en `src/images`
- Render estatico (sin SSR) configurado en `astro.config.mjs`

## Deploy con Docker + Nginx

Este directorio incluye `Dockerfile` y `nginx.conf` para despliegue en Coolify o Docker.
